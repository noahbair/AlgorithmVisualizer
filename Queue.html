Queue Visualization

What is a Queue?

A queue is a linear data structure that follows the First in First Out (FIFO) characteristic. The means that the first element that was added to the queue, is the first element to be removed. A queue can be implemented in a few ways, the most common implementations are those using an array or a list. For this visualizer, the queue was implemented using an array.

Main Operations:
- Enqueue: Used to add an element to the end of the queue.
- Dequeue: Used to remove an element from the front of the queue.
- Peek: Used to return the element in the front of the queue.
