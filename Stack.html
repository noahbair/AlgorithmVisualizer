Stack Visualization

What is a Stack?

A stack is a linear data structure that follows the Last in First Out (LIFO) characteristic. The means that the last element that was added to the stack, is the first element to be removed. A stack can be implemented in a few ways, the most common implementations are those using an array or a list. For this visualizer, the stack was implemented using an array.

Main Operations:
- Push: Used to add an element to the top of the stack.
- Pop: Used to remove an element from the top of the stack.
- Top: Used to return the element on the top of the stack.
