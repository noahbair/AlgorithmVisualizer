What is an Array?

An array is a collection / container of items stored at contiguous memory locations. When using an array remember that the type of data stored should be the same. For this implementation, we will be storing numeric data. You should note that arrays have what is called zero indexing. In the image below you can see the index values labeled in red. When deleting from your array remeber to insert the index value you wish to delete. For example if you wanted to delete the 3 you would place a 1 into the box located to the left of delete. For this visualization the array is created to hold only 10 elements. If this is exceeded an alert is displayed. If you recieve a pop up alert click the button again to dismiss the alert.
